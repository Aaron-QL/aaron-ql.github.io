  • 组成

    • 用户代理

DNS（因特网的目录服务

  • 构成

    • 一个由分层的DNS服务器实现的分布式数据库

      • 根DNS服务器

      • 顶级域DNS服务器

      • 权威DNS服务器

    • 一个使得主机能够查询分布式数据库的应用层协议

    • 本地DNS服务器（默认服务器

  • 运行在UDP之上，53端口

  • 查询方式

    • 递归查询

    • 迭代查询

  • DNS缓存

  • DNS记录

    • Name

    • Value

    • Type

      • A

        • Name:主机名

        • Value:IP地址

      • NS

        • Name是个域（foo.com

        • Value是个知道如何获得该域中主机IP地址的权威DNS服务器的主机名

      • CNAME

        • Name是主机名

        • Value是Name对应的规范主机名

      • MX

        • Name是别名

        • Value是别名为Name的邮件服务器的规范主机名

    • TTL

  • DNS报文

    • 12字节的首部区域

    • 问题区域

      • 名字字段

      • 类型字段

    • 回答区域

    • 权威区域

    • 附加区域
